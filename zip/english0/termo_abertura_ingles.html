Project: HEIL Games 4 Development for Web

Sponsor: PROF. ME. ELENILTON DEZENGRINI

Project Manager: Self-managed team

Date: 08/31/2024

Project Overview

The project for developing the Connect 4 game for the web aims to create an online version of the classic board game, allowing players to compete against each other in a digital environment.

Project Objectives

  • Develop an online version of the Connect 4 game to run on web browsers.
  • Allow human players to compete in real-time.
  • Ensure an intuitive and responsive interface for a pleasant gaming experience.

Project Scope

The project scope includes:

  • Development of the necessary front-end and back-end for the game's functionality.
  • Implementation of a player match management system.
  • Testing and validation of the system to ensure the game's stability and usability.

Estimated Budget

The total estimated budget for the project is R$ 100,000.00.

Estimated Timeline

The project will have an estimated duration of 8 hours, with the following key phases:

  • Planning Phase: 1 hour
  • Development Phase: 3 hours
  • Testing Phase: 1 hour
  • Deployment Phase: 1 hour
  • Final Presentation Phase: 5 min
