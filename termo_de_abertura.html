Projeto: Desenvolvimento do HEIL Games 4 para Web

Patrocinador: PROF. ME. ELENILTON DEZENGRINI

Gerente do Projeto: Equipe autogerenciável

Data: 31/08/2024

Visão Geral do Projeto

O projeto de desenvolvimento do jogo Ligue 4 para web visa criar uma versão online do clássico jogo de tabuleiro, permitindo que jogadores disputem partidas entre si em um ambiente digital.

Objetivos do Projeto

  • Desenvolver uma versão online do jogo Ligue 4 para rodar em navegadores web.
  • Permitir que jogadores humanos possam competir entre si em tempo real.
  • Garantir uma interface intuitiva e responsiva para uma experiência de jogo agradável.

Escopo do Projeto

O escopo do projeto inclui:

  • Desenvolvimento do front-end e back-end necessários para o funcionamento do jogo.
  • Implementação de um sistema de gerenciamento de partidas entre jogadores.
  • Testes e validação do sistema para garantir a estabilidade e usabilidade do jogo.

Orçamento Estimado

O orçamento total estimado para o projeto é de R$ 100.000,00.

Cronograma Estimado

O projeto terá uma duração estimada de 8 horas, com as seguintes fases principais:

  • Fase de Planejamento: 1h
  • Fase de Desenvolvimento: 3h
  • Fase de Testes: 1h
  • Fase de Implantação: 1h
  • Fase de Apresentação Final: 5 min
